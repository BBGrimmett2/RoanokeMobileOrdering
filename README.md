## Editing Installation
Requirements:
* Node.js
* Expo

### From CLI:
1. Clone Repository:
```
git clone https://github.com/BBGrimmett2/RoanokeMobileOrdering.git
```

2. Install Libaries:
```
npm install
```

3. Expo start:
```
expo start
```
