Github in VSCode

Set Up:
Install Git
https://git-scm.com/downloads

Install Extension in VSCode (Github Pull Request and Issues)
https://code.visualstudio.com/docs/editor/github

Install Github Desktop
https://desktop.github.com/

Issues: 
Error loading webview: Error: Could not register service workers: TypeError: Failed to register a ServiceWorker for scope
--> Clear Cache
https://stackoverflow.com/questions/67698176/error-loading-webview-error-could-not-register-service-workers-typeerror-fai

Creating branch:
-- Ctrl+Shift+P "Git: Create Branch"
-- Under Source Control Tab you can control the branches.

Pull and Push Edits:
    Push:
    -- Under Source Control you can view changes made since last push 
    -- Hit check mark and type a commiut message in textbox that appears
    -- Under Source Control sync changes

    Pull:
    -- Under Source Control go to the three dot menu
    -- Go to "Pull, Push" 
    -- "Pull From" and choose the branch you wish to pull 

    Merge to main:
    -- Under Source Control go to the three dot menu
    -- Go to "Branch" 
    -- "Merge Branch"
    -- Click which branch to merge to
